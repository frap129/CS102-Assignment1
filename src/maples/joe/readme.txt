README
------

How to use:
1. Make sure all *.java files from the zip are in the same directory
2. Open Assignment1.java in jGrasp
3. From the top ribon, select "Build", then "Run Arguments"
4. Enter the full path to your input file. 
   On Windows, hold ctrl+shift and right click to reveal the "Copy as path" option.
   If using this option, remove the quotes after pasting into run arguments.
5. Click the run button.
6. The menu will guide you on using the options provided.
